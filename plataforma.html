Manejo con API

Así viajaría cada respuesta automática

Webhook Meta envía el mensaje

Instagram, Facebook, Messenger o WhatsApp notifican tu servidor cuando entra una conversación.

Backend Tu servidor decide

Valida el canal, guarda la conversación, consulta la base entrenada y aplica reglas de seguridad.

Bot Se genera la respuesta

El motor busca una respuesta aprobada o usa IA con contexto del negocio y límites definidos.

Graph API Se responde al cliente

El servidor envía la respuesta por el endpoint oficial del canal correspondiente.
